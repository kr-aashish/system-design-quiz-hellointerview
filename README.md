# System Design Quizzes

A collection of interactive system design quizzes built with React and Vite.

## Available Quizzes
- Contention Quiz
- Handling Large Blobs Quiz
- Multi-Step Processes Quiz
- Real-Time Updates Quiz
- Scaling Reads Quiz
- Scaling Writes Quiz

## Development Setup

To run locally, execute:

```bash
npm install
npm run dev
```

Visit [http://localhost:5173/](http://localhost:5173/) to view the index.
